HTML / NAVIGATION APIs
================================================================================

--------------------------------------------------------------------------------
GET / — Interactive Map
--------------------------------------------------------------------------------
The main map viewer. All parameters are optional query strings.

Parameter       Type     Description
---------       ----     -----------
sector          string   Sector name or T5SS abbreviation
hex             string   Hex location (XXYY)
scale           number   Pixels per parsec (default: 64)
style           string   poster, print, atlas, candy, draft, fasa, terminal, mongoose
options         number   Rendering options bitmask (see below)
q               string   Search query
sx,sy,hx,hy    number   Sector/hex coordinates
x,y             number   Map-space coordinates
hideui          number   Hide UI elements (1=on)
routes          number   Toggle routes overlay (0=off)
rifts           number   Shade rifts (0=off)
nogrid          number   Hide hex grid (1=on)
po              number   Population overlay (1=on)
im              number   Importance overlay (1=on)
dw              number   Droyne world overlay (1=on)
an              number   Ancient world overlay (1=on)
mh              number   Minor race homeworld overlay (1=on)
stellar         number   Stellar representation (1=on)
hw              string   Highlight worlds by characteristic
ew              string   Empress Wave overlay (date format)
dimunofficial   number   Dim unofficial sectors (1=on)
review          number   Tint by review status (1=on)
sscoords        number   Subsector-style hex numbering (1=on)
allhexes        number   Number all hexes (1=on)
galdir          number   Toggle galactic directions overlay (0=off)
forceui         number   Show UI in iframe (1=show)
p               string   Compact coordinates (x!y!s format)
marker_*        various  Custom marker placement (sector, hex, sx, sy, hx, hy, x, y, url)
yah_*           various  "You Are Here" placement (sector, hex, sx, sy, hx, hy, x, y)
ox,oy,ow,oh,os  various  Overlay rectangle (map-space coords + CSS color)
or              string   Compact overlay rectangles (x!y!w!h!color~...)
ocx,ocy,ocr,ocs various  Overlay circle (center x, center y, radius, CSS color)
oc              string   Compact overlay circles (x!y!r!color~...)

Returns: HTML interactive map page

Example: https://travellermap.com/?sector=Spinward%20Marches&hex=1910&scale=64&style=poster

--------------------------------------------------------------------------------
GET /api/search — Search Worlds & Regions
--------------------------------------------------------------------------------
Full-text world/region search with wildcards and field-scoped operators.

Parameter   Type    Required   Description
---------   ----    --------   -----------
q           string  Required   Query string
milieu      string  Optional   Milieu code
accept      string  Optional   Output format (text/xml for XML)
jsonp       string  Optional   JSONP callback name

Query operators:
  exact:     Exact match
  like:      Phonetic match
  uwp:       UWP field match
  pbg:       PBG field match
  zone:      Zone field match
  alleg:     Allegiance field match
  stellar:   Star type match
  remark:    Remark requirement
  in:        Scope to sector name

Wildcards:
  * or %     Zero or more characters
  ? or _     Exactly one character
  [a-z]      Character range

Returns: JSON { Results: [...] } or XML

Example: https://travellermap.com/api/search?q=Regina
Example: https://travellermap.com/api/search?q=in:spin+uwp:A

--------------------------------------------------------------------------------
GET /api/route — Jump Route Planning
--------------------------------------------------------------------------------
Finds a jump route between two worlds using pathfinding.

Parameter   Type    Required   Description
---------   ----    --------   -----------
start       string  Required   Start location (SECTOR HEX or world name)
end         string  Required   End location (SECTOR HEX or world name)
jump        number  Optional   Max jump distance 0-12 (default: 2)
wild        number  Optional   Wilderness refueling only (1=yes)
im          number  Optional   Imperium members only (1=yes)
nored       number  Optional   Avoid red zones (1=yes)
aok         number  Optional   Allow anomalies (1=yes)
x           number  Optional   Search position X (world-space)
y           number  Optional   Search position Y (world-space)
milieu      string  Optional   Milieu code
accept      string  Optional   Output format (text/xml for XML)

Returns: JSON route path object, or HTTP 404 if no route exists

Example: https://travellermap.com/api/route?start=spin+1910&end=spin+2110&jump=2

GET /api/poster
GET /data/{sector}/image
GET /data/{sector}/{subsector}/image
GET /data/{sector}/{quadrant}/image
POST /api/poster — Sector / Subsector / Region Poster
--------------------------------------------------------------------------------
Renders a high-quality map of a sector, subsector, quadrant, domain, or
custom bounding box.

Parameter   Type    Required   Description
---------   ----    --------   -----------
sector      string  Optional   Sector name
subsector   string  Optional   Subsector A-P or name
quadrant    string  Optional   Alpha, Beta, Gamma, Delta
domain      string  Optional   Domain: Sylea, Vland, Gateway, Ilelish,
                               Antares, Sol, Deneb
x1,y1       number  Optional   Top-left corner (world-space) for custom box
x2,y2       number  Optional   Bottom-right corner (world-space) for custom box
scale       number  Optional   Pixels per parsec (default: 64)
rotation    number  Optional   Rotate 0-3 (90 degree increments)
hrotation   number  Optional   Hex rotation in degrees (60 or -60 useful)
clampar     number  Optional   Clamp aspect ratio (1=yes)
options     number  Optional   Rendering options bitmask
style       string  Optional   Visual style
milieu      string  Optional   Milieu code
lint        number  Optional   Fail on warnings (1=yes)
thumb       number  Optional   Render thumbnail (1=yes)
compositing number  Optional   Transparent background (1=yes)
accept      string  Optional   Output format (application/pdf, image/svg+xml)

POST body: Raw sector data (Content-Type: application/x-www-form-urlencoded,
           multipart/form-data, or text/plain)

Returns: PNG image (or PDF/SVG via accept parameter)

Example: https://travellermap.com/api/poster?sector=spin&subsector=C&style=atlas&scale=48

================================================================================
COMMON NOTES
================================================================================

Output format:
  Default output is JSON.
  Pass accept=text/xml or HTTP header Accept: text/xml for XML output.
  Rendering endpoints accept application/pdf or image/svg+xml for vector output.

JSONP:
  Append &jsonp=callbackName to supported data endpoints.

CORS:
  CORS headers are present. Browser fetch() / XMLHttpRequest works directly
  without a proxy.

Location parameter groups (use one group per request):
  - sector name:    sector=, hex=, subsector=
  - sector coords:  sx=, sy=, hx=, hy=
  - world-space:    x=, y=

REST-style aliases:
  Many endpoints have clean REST URL aliases under /data/... in addition to
  the query-string form under /api/...

  /api/sec?sector=X         == /data/X
  /api/metadata?sector=X    == /data/X/metadata
  /api/msec?sector=X        == /data/X/msec
  /api/jumpworlds?...       == /data/{sector}/{hex}/jump/{jump}
  /api/poster?sector=X      == /data/X/image
  /api/jumpmap?...          == /data/{sector}/{hex}/jump/{jump}/image
